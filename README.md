# VITYA MoldSim

Injection molding design tool for Michigan manufacturers.

## Features
- Cost estimation using MoldSim physics
- Cooling analysis with Cross-WLF viscosity model
- DFM checking for manufacturability
- Material database with ABS, PP, PE-HD, PA6, PC
